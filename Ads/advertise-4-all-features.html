gControl is a Cloud Security, Control & Identity Platform for Enterprise. gControl is a powerful tool to complement for Google apps and help you reap the maximum benefits from Google Apps.

Armed with powerful security features like IP Restriction, Device Restriction, Session timeout, Time Restriction Consumer Gmail Block gControl makes sure that your data is safe on the cloud. gControl comes with some cloud security essentials to enhance the productivity of your cloud application. Landing Pages, Custom login page and Landing Broadcast feature of gControl takes care of your Google apps essentials.

Login Features

Custom Login Page

gControl helps you design customized login pages for Google Apps for free . It provides Google Apps Admin the ability to set the custom login screen with SSO for Google Apps.

Landing Page

gControl comes up with another attractive feature of Landing pages. Make sure the concerned information reached the right person with Custom landing pages.

Agreement

gControl provides you with the agreement policy which helps in signing the one time acceptance agreement between the employee and the organization.

Login Broadcast

Broadcast message in cloud based application helps in text advertising in displaying important messages in cloudbased application.
Identity Features

Password Policy

gControl helps in adding a strong password policy for your google Apps, quite similar to MS active directory. The policies can be implemented for individuals as well as Organizational Units.

2 Factor Authentication

Increase cloud Security by enforcing 2 Factor authentication on your and organizational units. Use Google Authenticator for Token generation.

Forgot Password

Google Apps admin can define some predefined questions and set the policies to have min questions answered by the user to allow them to reset their own passwords.

AD Integration

gControl will provide integration with Microsoft Azure Active Directory for single sign on authentication. Single Sign on solution will reduce maintenance of multiple credentials and data overload.
Control Features

Browser Restriction

Enforce users to use only one browser which is easy to manage centrally, Google Chrome is preferred choice of Admin as it can be managed centrally and extension / apps can be pushed.

IP Restriction

IP restriction Feature of gControl enables Admin to define policies for a particular OU or a subset of user and restrict access by providing controlled access to predefined IP Address.

Time Restriction

Restrict your users from accessing business email only in specific slots in a day with gControl's Time Restriction policies. Time Restriction allows Google apps admin to define policies for Subset of users or Organizational Unites.

Device Restriction

Device restriction is the most important cloud firewall solution for any corporate organization. Restrict your employees from accessing corporate data from any other device except the allotted corporate device.

Session Timeout

Enforce session timeout for Google Apps to enhance the security around the Google Apps usage.

Consumer Gmail Block

Block personal Gmail while allowing Google Apps for Business to ensure that the employees focus at work.

3rd Party App Block

gContol helps to blacklist 3rd party application to ensure data security. Prevent your employees to sign into 3rd party application by blacklisting them.
